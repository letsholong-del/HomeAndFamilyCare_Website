# Home and Family Care Website

Customer-facing static website for Home and Family Care.

## What this site includes
- Conversion-focused landing page for home cleaning and babysitting
- Custom hero, cleaning and babysitting image assets
- Focused service descriptions for home cleaning and babysitting
- Detailed domestic cleaning checklist
- Trust proof section covering checks, insurance-aware process and quality follow-up
- Cleaning guide pricing from R300
- Babysitting quotation request flow
- Testimonials-ready section for real customer reviews
- WhatsApp quote handoff using `061 225 2597`
- Helper application handoff for cleaners and babysitters
- Trust, safety, differentiation and FAQ content
- GitHub Pages deployment workflow
